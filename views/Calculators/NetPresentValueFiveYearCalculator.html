Net Present Value of an Investment for a Five Year Period Calculator

This equation is used to determine the net present value of an investment made today. This is a method which will determine if a project will be profitable or not.

Equation

\(Net\ Present\ Value = \Big[{\frac{ Yr1 Cash Flow }{(DiscountRate)}}\Big] \ + \Big[{\frac{ Yr2Cash Flow }{(DiscountRate) ^ 2}}\Big] \ + \Big[{\frac{ Yr3Cash Flow }{(DiscountRate) ^ 3}}\Big] \ + \Big[{\frac{ Yr4Cash Flow }{(DiscountRate) ^ 4}}\Big] \ + \Big[{\frac{ Yr5Cash Flow }{(DiscountRate) ^ 5}}\Big] - Initial Investment \)
